IFC Advisory Projects Analysis Dashboard

📌 Project Overview
This project presents an interactive Power BI dashboard analyzing IFC (International Finance Corporation) advisory projects across multiple countries, regions, and business lines.
The dashboard provides insights into investment trends, project distribution, and performance metrics from 2009 to 2026.

🎯 Objectives

Analyze total investments across countries and regions

Understand project distribution by status (Active, Completed, Hold)

Identify high-investment business lines

Track investment trends over time

Provide a global view of IFC advisory projects

📂 Dataset Description

The dataset includes:

Project Name

Country & Region

Investment Amount (2009–2026)

Business Line

Project Status (Active, Completed, Hold, Pending)

Approval Year

📈 Key Insights

💰 Total Investment: 3 Billion+

📊 Total Projects: 2089

✅ Majority of projects are Completed (~63%)

🌍 Investments are spread across multiple global regions

🏗️ Highest investments observed in Environment & Social Sustainability and Corporate Advisory
